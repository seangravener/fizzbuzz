# A FizzBuzz Solution

## Exercise Instructions:
Print integers 1 to N, but print “Fizz” if an integer is divisible by 3, “Buzz” if an integer is divisible by 5, and “FizzBuzz” if an integer is divisible by both 3 and 5.

## To run

In terminal:
```bash
node src/index.js
```

In browser:
Open index.html
